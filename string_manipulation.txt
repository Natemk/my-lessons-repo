What is a Dictionary?

A dictionary stores data as:

Key ----> Value
Example

student = {
    "name": "John",
    "age": 20,
    "grade": 85
}

name -----> John
age  -----> 20
grade-----> 85

Why not use a list?

Bad:

student = [
    "John",
    20,
    85
]

what does this mean:

student[2]

---------------------------------------------

Good:

student = {
    "name": "John",
    "age": 20,
    "grade": 85
}

Now 

student["grade"]
-------------------------------------------------------------------------------

using get()

Bad:

print(student["salary"])

Output = KeyError

Good:

print(
    student.get("salary", "not found")
)

Output : not found
---------------------------------------------------------------------------

looping though keys

student = {
    "name": "John",
    "age": 20,
    "grade": 85
}

for key in student:
    print(key)

----------------------------------------------------------------------------=--

looping though Values

student = {
    "name": "John",
    "age": 20,
    "grade": 85
}

for value in student.values():
    print(value)

-------------------------------------------------------------------------------
looping through both values and keys

student = {
    "name": "John",
    "age": 20,
    "grade": 85
}

for key, value  in student.items():
    print(key, ":", value
    )
