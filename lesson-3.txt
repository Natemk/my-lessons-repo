User Input and Interaction

At this point the student knows:

✅ print()

✅ Variables

✅ Strings

✅ Numbers

✅ Booleans

The next logical step is teaching the computer to receive information from the user.

Learning Objectives

By the end of this lesson students should be able to:

Understand what input is
Use the input() function
Store user input in variables
Display user input
Build simple interactive programs
Lesson Structure (90 Minutes)
Part 1: Review Previous Lesson (10 Minutes)
Ask

What is a variable?

Expected Answer:

A container that stores information

What is a string?

Expected Answer:

Text

What is an integer?

Expected Answer:

A whole number
Part 2: What is Input? (10 Minutes)
What to Say

"So far our programs only talk."

Example:

print("Hello")

Output:

Hello

The computer speaks.

But what if we want the user to speak back?

We use:

input()
Part 3: First Input Program (15 Minutes)

Type:

name = input("What is your name? ")

print(name)

Run it.

Example:

What is your name? Nathaniel
Nathaniel

Explain:

input()

means:

Wait for the user to type something
Part 4: Making It Friendly (10 Minutes)

Modify:

name = input("What is your name? ")

print("Hello", name)

Example:

What is your name? Nathaniel
Hello Nathaniel
Explain

The program now:

Asks a question
Waits
Receives input
Responds
Part 5: Multiple Inputs (15 Minutes)

Show:

name = input("What is your name? ")
country = input("Which country are you from? ")

print("Hello", name)
print("You are from", country)

Example:

What is your name? Nathaniel
Which country are you from? Canada

Hello Nathaniel
You are from Canada
Part 6: Creating a Profile (10 Minutes)

Show:

name = input("Name: ")
age = input("Age: ")
city = input("City: ")

print("----- PROFILE -----")
print("Name:", name)
print("Age:", age)
print("City:", city)

Output:

----- PROFILE -----
Name: Nathaniel
Age: 15
City: Hamilton
Part 7: Important Concept – Everything Is Text (10 Minutes)

This is the first major programming lesson.

Show:

age = input("Enter your age: ")

print(age)

Ask:

What type is age?

Answer:

String

Always.

Even if the user enters:

15

Python still treats it as:

"15"
Demonstration

Show:

age = input("Enter age: ")

print(age + age)

User enters:

15

Output:

1515

Ask:

Why didn't we get 30?

Because Python sees:

"15" + "15"

not

15 + 15
Part 8: Converting Input to Numbers (5 Minutes)

Introduce:

int()

Show:

age = int(input("Enter age: "))

print(age + age)

Input:

15

Output:

30
Part 9: Mini Project (10 Minutes)

Build an Age Calculator

name = input("What is your name? ")
age = int(input("How old are you? "))

future_age = age + 10

print("Hello", name)
print("In 10 years you will be", future_age)

Example:

What is your name? Nathaniel
How old are you? 15

Hello Nathaniel
In 10 years you will be 25
